

ESTRUCTURA DEL PROYECTO

La estructura del sitio es la siguiente:

proyecto-web/
│
├── index.html                     → Página principal
│
├── pages/
│     ├── arquitectura.html        → Arquitectura Cliente-Servidor
│     ├── lenguaje-marcas.html     → Ejemplos HTML5
│     ├── css3.html                → Reglas de CSS3 y animaciones
│
├── assets/
│     ├── css/
│     │     └── styles.css         → Estilos generales
│     │
│     ├── js/
│     │     └── main.js            → Funcionalidades interactivas
│     │
│     └── img/                     → Imágenes del proyecto
│
└── docs/
      ├── instrucciones_instalacion.md
      └── estructura_proyecto.md

DESCRIPCIÓN DE CADA SECCIÓN
★ 1. Página de inicio (index.html)

Contiene:

Encabezado principal

Barra de navegación

Presentación del sitio

Botones de acceso a las demás secciones

Pie de página

★ 2. Arquitectura de Servicios (arquitectura.html)

Explica:

Qué es la arquitectura cliente-servidor

Cómo funciona la comunicación entre navegador y servidor

Recursos gráficos que ilustran el proceso

★ 3. Lenguaje de Marcas HTML5 (lenguaje-marcas.html)

Incluye:

Ejemplos de etiquetas HTML5

5 elementos de control funcionando:
✔ input
✔ select
✔ textarea
✔ button
✔ checkbox / radio

Demostración práctica del uso de etiquetas semánticas

★ 4. CSS3 y Animaciones (css3.html)

Incluye:

Reglas CSS3 aplicadas a elementos HTML

Ejemplo de animación CSS3

Uso de clases reutilizables

Uso de variables, sombras, bordes redondeados, transiciones

RELACIÓN ENTRE LAS SECCIONES

Todas las páginas comparten:
✔ Una barra de navegación
✔ Una estructura semántica coherente
✔ Mismo archivo CSS
✔ Mismo script JS

La navegación crea una arquitectura clara tipo:

Inicio → Arquitectura → Lenguaje de Marcas → CSS3 → Regresar a Inicio
